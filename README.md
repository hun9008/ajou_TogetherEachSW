# Commit Rule

- main branch 하나만 사용.
- main의 README.md에는 각 주에 공부한 내용 요약.
- 개인 폴더에 각자 방식으로 매주 공부한 내용 정리.
- 각자 공부한 자료 매주 일요일까지 업로드.
- 다른 사람 폴더 수정 금지.
